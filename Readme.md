
# Project Name

DigiTools Platform



# Description

DigiTools is a modern digital marketplace landing page designed to sell premium tools and templates. It is built using React and features an interactive product listing and dynamic cart system. The project uses a responsive UI, pricing tables, and a clean 3-step onboarding process to simplify the user experience.



# Technology used

Html
Tailwind CSS, DaisyUI
JavaScript
React.js
React-Toastify (NPM Package)
JSON (For Product Data)



# 3 Features about this project

1. Interactive Shopping Experience
   Users can browse through a collection of digital products, add items to the cart, view selected tools, and remove them with instant updates.

2. Fully Responsive and Clean UI
   The interface is designed to work smoothly across all devices, featuring a modern layout with sections like hero banner, stats, pricing, and step-by-step onboarding.

3. Smooth User Flow with Feedback
   The application provides clear navigation, visual feedback using toast notifications, and an easy-to-use checkout experience that enhances usability and engagement.
